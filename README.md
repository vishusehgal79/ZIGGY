# Ziggy - Food Ordering Website 🍔

## Overview

Ziggy is a responsive food ordering website where users can browse different food categories and view food items. The project focuses on creating a clean user interface and smooth navigation using basic web technologies.

## Features

* Browse different food categories
* View food items and details
* Responsive design for different screen sizes
* Simple and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in any web browser.
